Posicinamiento CSS

El navegador coloca cada elemento teniendo en cuenta el orden en el que aparece en el documento y su tipo de visualización dependiendo si un elemento es de línea o bloque

Con FLOAT y POSITION podemos modificar este comportamiento.

Elementos flotantes

  • float: Convierte un elemento en flotante desplazandolo hasta la zona mas izquiera o derecha de la posición original.
  • clear: Limpia la flotación (left, right, both)

Tipos de posicinamiento

  • static(default)
  • relative
  • obsolute
  • fixed
  • sticky

Para mover los elementos posicionados se activan las propiedades:

  • top (vertical-eje Y)
  • bottom (vertical-eje Y)
  • left (horizontal-eje X)
  • right (horizontal-eje X)
  • z-index (profundidad -eje Z)

Las propiedades top y left van a tener preferencia por sobre bottom y right respectivamente.
Estas 5 propiedades no funcionan con el valor estatic

VOLVER AL INDICE

relative

El elemento conserva sus dimensiones y posición original en el flujo del documento. Si se mueve toma como referencia su posición original

absolute

El elemento pierde sus dimensiones y posición original en el flujo del documento. Si se mueve puede tomar como referencia 2 elementos:

  1. El primer ancestro con posicionamiento relativo
  2. Si no encuentra un ancestro relativo, se mueve respecto al documento HTML

fixed

El elemento pierde sus dimensiones y posición original en el flujo del documento. Si se mueve toma como referencia el documento html y queda fijo en esa posición, si se hace scroll el elemento no se moverá.

sticky

Para poder enterderlo se podría decir que es la combinación de posicionamiento relative y fixed, ya que las propiedades top, left, rigth y bottom no mueven el elemento si no que establece un punto de referencia y mientras no llegue a ese valor el elemento se comporta como un relative y cuando llega se convierte en fixed. Para que este comportamiento funcione el elemento sticky debe ser hijo directo del body o su elemento contenedor debe de tener las dimensiones establecidas y solo será sticky dentro del contenedor padre.

z-index

Esta propiedad nos sirve para dar orden a la superposición de elementos, es decir si dos elementos estan uno encima del otro, con la propiedad z-index podemos decidiar que elemento queremos que vaya delante de otro. esto es posible porque el z-index recibe como valor un numero y dependiendo de si el numero es mas grande o mayor que el resto, este elemento se pondrá primero. (recomendable usar estos numeros de 10 en 10 o si es un proyecto grande de 100 en 100 para que cuando aparezcan elementos que quisieras poner entre un rango lo puedas hacer normal)
