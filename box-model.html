Modelo de caja en CSS

Es la forma en que CSS ve a los elementos HTML, los ve como cajas con las siguientes propiedades:

  1. Contenido (content): generalmente es texto
  2. Borde (border)
  3. Relleno (padding): distancia interna entre el borde y contenido
  4. Margen (magin): distancia externa entre el borde y el body entre otros elementos.
cada elemento tiene 4 lados y podemos manipular todos los lados individualmente, estos son:
  1. Arriba (top)
  2. Abajo (buttom)
  3. Derecha (rigth)
  4. Izquierda (left)

Cajas de linea y cajas de bloque

Línea
  • Ocupan solo el espacio necesario para mostrar su contenido
  • No tienen dimensiones modificables (alto y ancho)
  • Permiten otros elementos a su lado
  • Padding y Margin solo empujan a elementos en horizontal, NUNCA VERTICAL

Bloque

  • Ocupan todo el ancho disponible, generando asi un salto de línea
  • Tienen dimensiones modificables
  • No permiten otros elementos al lado (Aunque se les asigne un ancho o se les sube el margin)

Propiedad display

La propiedad display nos permite establecer un comportamiento de visualización del elemento, este puede ser ocultar un elemento(none), cambiar a un elemento de bloque a uno de linea (inline) y viceversa(block), hacer que un elemento ya sea de linea o bloque; puedan compartir propiedades (inline-block) como por ejemplo ponerle un margin y padding arriba y abajo a los elementos de tipo linea. La manera de usarlo es: 

                
        .detalles{
            display: inline-block;
        }

Propiedad Visibility

Se usa para ocultar elementos pero consevando el espacio que ocupan, a diferencia del display: none; que nos oculta el elemento haciéndolo como si no existiera, la forma de usarlo es: 

                
        .detalles{
            Visibility: hidden;
        }

Propiedad Overflow

Esta propiedad define como se muestra el contenido desbordado, tanto en el eje horizontal como en el eje vertical. por defecto su valor es visible por lo tanto el contenido desbordado aun es visible y puede se molesto al tapar a otros elementos debajo o de sus lados, para ocultarlo se usa hidden con eso el contenido que esta fuera de las dimensiones establecidas se ocultan, entonces para poder acceder a dicho contenido se agrega el valor scroll, esto nos agregara una barra de desplazamiento vertical. Tamaño de caja

El tamaño de caja inicialmente es el tamañano que el content, asi que si nosotros establecemos un tamaño para nuestro elemento, este será el tamaño que tomará content pero si le agregamos padding y bordes al elemento, crecerá el tamaño total de la caja, dejando al content con el valor que establecimos. Esto podria causar inconvenientes a la hora de maquetar nuestra web. Asi que para evitar eso, podemos modificar para que la caja se ajuste automáticamente al tamaño que nosotros queremos establecer, quiere decir que el valor de dimensiones que nosotros establezcamos no se aplicará solo al content si no la caja entera. Esto se logra gracias a la propiedad: box-sizing: border-box;

Ahora al momento de maquetar una página, colocar la propiedad a cada elemento o clase seria un poco tedioso, entonces para aplicarlo de manera general a todo el documento HTML, podemos usar la técnica que nos lo propone Paul Irish: 

                
    html {
        box-sizing: border-box;
        }
        *, 
        *:before,
        *:after {
        box-sizing: inherit;
        }
Esto podemos colocarlo en cualquier lugar (recomendable en las primeras lineas) de nuestra hoja de estilo ya que aplicará a todo elemento gracias a los Pseudoelementos :before y :after
