Variables y Funciones

Custom Propierties (Variables CSS)

Las variables son definidas de la siguiente manera --name-variable: valor; la variable puede almacenar distintos valores como: colores, medidas, funciones, etc. Las variables solo funcionan el propio selector donde se declaro la variable y en los hijos del elemento selector donde se declaró la variable. Pero es posible crear variables que funcionen en cualquier selector, eso se logra creando las variables en la pseudoclase :root{ --variable: valor;} entonces, ahora sí podemos usar la variable en los selectores que queramos.

Un ejemplo:
