Slide 1

Para lograr el efecto del scroll actual, los elementos deben de estar en un contenedor principal, en este caso es la etiqueta main a este contenedor se le dará un ancho y alto completo del viewport, ademas se le agrega las propiedades overflow-y: scroll; y scroll-snap-type: block mandatory; Es este ultimo el ecargado de hacer este efecto en el scroll

Slide 2

Dentro del contenedor main hay varios contendores section dentro de cada section esta un ultimo contenedor que es article, en este contenedor están los elementos que se quiera mostrar en el viewport actual, en este caso son el titulos y estas explicaciones.
