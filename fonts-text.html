Cargar fuentes externas

De manera local

Para cargar fuentes externas debemos tener la fuente en formato woff (Web Open Font Format ) o woof2 en nuestra carpeta de assets y otro que tengas. Para poder usar la fuente tenemos que usar el @font-face: 

            
    @font-face{
        font-family: "Fontname";
        src: url("ruta/archivo.woff") format("woff");
        font-display: swap;
    }
Para usarlo solo usas el nombre que le diste como valor al font-family en el @font-face. Cuando el recurso de la fuente esta en un servidor entonces es conveniente usar la la propiedad: font-display que tiene como valores lo siguiente:
  • auto - Permite que el navegador utilice el método predeterminado que suele ser block.
  • block - Oculta brevemente hasta que la fuente se haya descargado por completo
  • swap - Indica al navegador que utilice la fuente alternativa para mostrar el texto hasta que la fuente personalizada se haya descargado por completo
  • fallback - es una mezcla de auto y swap
  • optional - Ocultará el texto, luego lo cargará con la fuente alternativa y finalmente aplicará la fuente personalizada

Mas propiedades para texto

  • text-align
  • text-decoration
  • text-ident
  • text-overflow (requiere propiedad overflow: hidden; y white-space: nowrap;)
  • text-transform (para covertir el texto en mayusculas y minusculas)
  • letter-spacing (para el espaciado de cada caracter)
  • white-space (define como se maneja el espacio en blanco del elemento)
  • word-break (Define como debe dividirse las palabras al llegar al final de una linea)
  • word-spacing (Para el espaciado de palabras)
  • writing-mode (horizontal, vertical)
