PROPIEDADES QUE AFECTAN A LOS HIJOS DEL CONTENEDOR

flex-grow

Flex-grow es una propiedad que dota a los elementos hijos con la capacidad de crecer en el espacio sobrante del contenedor padre, su valor por defecto es 0, si ponemos 1 quiere decir que todos los elementos tendran un tamaño sumado la división del espacio sobrante entre 3 más su tamaño definido, esto quiere decir que si mi contenedor mide 1000px de ancho y los tres elementos hijos miden 100px de ancho, el espacio sobrante seria 700px, ahora al momento de aplicarle el flex-grow: 1; a todos los elementos hijos, dividimos el espacio sobrante (700px) entre la cantidad de elementos hijos (3), esto nos daria un total 233.33px, este es el tamaño que crecerá los elementos hijos para ocupar todo el espacio del contenedor padre

Cuando a todos los hijos se les da el mismo valor

Cuando a uno de ellos se les da el doble que a los demas

Entonces si a uno de los hijos le aplicamos el flex-grow con valor 2, quiere decir que ese elemeto hijo tomará el doble de espacio sobrante que sus demas hermanos, esto quiere decir que ahora el espacio sobrante se tendrá que dividir entre 4, ya que uno de ellos tomará un espacio adicional más que sus hermanos

Flex-Shrink

Al contrario de flex-grow, esta propiedad dota de la capacidad para poder reducir su tamaño cuando su contenedor es muy pequeño, y posee las mismas caracteristicas que el flex-grow, solo que esta vez para reducir su tamaño, Por defecto esta propiedad ya viene activada flex-shrink:1; para desactivarla solo se tiene que darle como valor 0

Flex-basis

Esta propiedad nos sirve para establecer un ancho (si el eje de alinineacion de flex es horizontal) y alto si el eje es vertical. Su valor por defecto es auto, si cambiamos su valor, este hara que la propiedad width o height sean ignorados esten donde esten

Normal, en el eje horizontal

En el eje vertical + Shorthand (flex: flex-grow flex-shrink flex-basis;)

PROPIEDADES order Y align-self

La propiedad order en flex-box nos sirve para establecer un orden para los elementos de un contendor, la regla es: el valor menor, va primero, su valor por defecto es 0 y acepta valores negativos. La propiedad align-self nos sirve para alinear un elemento en concreto, esto hara que el align-items no aplique para ese elemento.
