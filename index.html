Selectores Basicos:

  1. Por etiqueta: 
                    
        h1{ 
            propiedad: valor;
        }
  2. Por ID: 
                        
        #id_etiqueta {
            propiedad:valor;
        }
  3. Por clases (recomendada): 
                        
        .class_etiqueta {
            propiedad:valor;
        }

Al <h1> le aplicamos estilo con su etiqueta, al titulo del temario por su id="" y al marcador de las lista por su class=""

Explicación con las listas de arriba, los selectores hijos se refiere al orden en que estan las etiquetas, los hijos son las etiquetas que estan despues de la etiqueta de referencia, que en este caso la etiqueta de referenciaseria <ul> </ul> y el hijo directo seria la etiqueta <li> </li>, entonces el hijo descendiente es aquella etiquetaque esta dentro de la etiqueta <li></li>, como se puede deducir dentro de cada <li> </li> esta la etiqueta <b></b>. (recordar que li es una etiqueta de bloque, y b es una etiqueta de linea).

Codigo Css para hijos directos
        
    .hijos_directos>li{
        background-color:aquamarine;
    }
Codigo Css para hijos descendientes
        
    .hijos_descendientes b{
        background-color:aquamarine;
    }

Explicación "hermanos" hace referencia a las etiquetas que estan en un mismo nivel, en los ejemplos tomamos a la etiqueta que contiene "Verano" para tomarlo como referencia para aplicar cierto estilo:

Codigo para hermanos en general: 

            
    .hermanos_general~li{
        background-color: aquamarine;
    }

Cuando se pone "~" Los estilos se aplican a toda etiqueta hermano que es ta por debajo del que se tomó por referencia.

Codigo para hermanos Adyadcentes: 

            
    .hermanos_adyacentes+li{
        background-color: aquamarine;
    }

Cuando se pone "+" Los estilos se aplican solamente a la etiqueta hermano que esta debajo del que se tomo por referencia.

Para hacer el ejemplo anterior use la etiqueta nav que es la que lleva el class principal, con esa clase podemos darle estilos a los elementos que se encuentren dentro de esa etiqueta, en el ejemplo tenemos una lista de enlaces que dimos estilos filtrando el valor de sus atributos, como por ejemplo si el target="_blank" entoces se aplica negrita a esos elementos. Existen varias formas de filtrar los elementos a darle estilos, algunos son

  • Con *: .selectores_atributo a[href*="jonmircha"]{} aqui le decimos que aplique el estilo a todo elemento que contenga la palabra "jonmircha" como valor del atributo href. Sin importar que hay antes o despues de dicha palabra.
  • Con ^: .selectores_atributo a[href^="http://"]{} Para aplicar los estilos a todo elemento que el valor del href inicie con "http://".
  • con $: .selectores_atributo a[href$=".com"]{} Lo mismo que el anterior pero, esta vez con el final del valor del href.
  • Con ~: .selectores_atributo a[class~="cursos"]{} Aqui usamos ~ para aplicar el estilo a todo elemento que el valor de class contenga "cursos " y ademas que este separada por espacios en una lista, eso quiere decir que cursos, no es el unico valor de class.
  • con |: .selectores_atributo a[class|="menu"]{} Usamos | para aplicar el estilo a todo elemento que contenga la palabra "menu" seguida de un guión medio, si la palabra "menu" existe como valor de un class, pero no lleva ningun guión medio seguidono se aplicará el estilo.

Agrupar Selectores

En el fomulario anterior se dio estilos a todos los elementos (excluyendo al botón), agrupando selectores, la manera de hacerlo es separandolos con comas, ejemplo: 

                
            .form-pseudoelementos input[type="text"],
            .form-pseudoelementos input[type="email"],
            .form-pseudoelementos textarea {
                border-radius: 10px;
                color: #00bb10;
                border-color: blue;
                font-size: 16px;
                background-color: antiquewhite;
                display: block;
                border-width: 2px;
                width: 200px;
                margin-bottom: 3px;
            }

El algoritmo de CSS

  • Cascada: Aplica estilos dependiendo del orden en la que se definen dichos estilos
  • Especificidad: Aplica estilos dependiendo el peso de un selector
  • Herencia: Es la capacidad de de un selector de obtener (heredar) los valores de sus ancestros más cercanos se usa inherit
